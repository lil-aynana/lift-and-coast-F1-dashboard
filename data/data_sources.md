# Data Sources for Lift and Coast Dashboard

This project uses the following datasets:

- Race results (finish positions, points)
- Qualifying results
- Driver info
- Constructor info
- Lap times (optional)
- Pit stop data (optional)
- Season schedule (race names & dates)

Datasets will be stored in:

- `data/raw/` → raw downloaded files
- `data/cleaned/` → cleaned and merged tables
